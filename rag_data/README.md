# RAG Data Folder

Put your grounding files here for Delulu.

Supported file types:
- `.md`
- `.txt`
- `.json`

Usage:
1. Add files into this folder.
2. Restart bot or run `/ragreload`.
3. Check `/ragstatus`.
4. Test retrieval with `/ragsearch <query>`.

Companion tips:
- Use `/remember <fact>` to store personal facts.
- Use `/aboutme` to view what Delulu remembers.
- Use `/companion` for quick usage guide.

Tips:
- Keep documents factual and concise.
- Split very large docs into smaller topical files.
- Add one file per topic for better retrieval quality.
